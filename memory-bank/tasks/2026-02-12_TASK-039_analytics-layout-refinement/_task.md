# TASK-039 — Analytics Layout Refinement

## Goal
Переработать вкладку аналитики под сценарий пользователя:
- сначала выбор опросника,
- аналитика по параметрам только при наличии 2+ оценок,
- кликабельный календарь для просмотра прошлых дней/периода,
- более эргономичный календарный блок (цвет в ячейке дня, без полноширинного дубля-графика).

## Linked Requirements
- `UR-034`, `UR-050`, `UR-051`: динамика и аналитика по вопросам/дням.
- `WF-008`: UI-изменения подтверждаются Playwright.

## Deliverables
- Единый analytics-панель фильтров (опросник + период + быстрые диапазоны).
- Упрощенный `ScoreChart`: календарь и summary дня без лишнего полноширинного бар-блока.
- `QuestionnaireStatsPanel`: показываются только вопросы с 2+ оценками; вопросы с недостатком данных вынесены отдельно.
- Обновленные RU/EN i18n ключи и coverage.
- Playwright артефакты нового UI.
