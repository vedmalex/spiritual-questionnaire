# Requirements (Task-007)

## Source
Пользовательский запрос в чате:

> пользователь может выходить и заходить в приложение, при выходе вся его информация выгружается в файл, для дальнейшего использования

## Acceptance
1. Есть действие logout в UI.
2. Logout автоматически экспортирует пользовательские данные в файл.
3. Экспортированный файл может быть импортирован для восстановления данных пользователя.
