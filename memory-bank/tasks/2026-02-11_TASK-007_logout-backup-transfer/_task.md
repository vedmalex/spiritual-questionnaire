# TASK-007 — Logout Backup Transfer

## Goal
Реализовать безопасный выход пользователя с автоматической выгрузкой его данных в файл и возможностью восстановления при следующем входе.

## Linked Requirements
- `UR-044`: При выходе пользовательские данные выгружаются в backup файл и могут быть восстановлены при повторном входе.

## Deliverables
- Кнопка выхода из приложения.
- Автоматическая выгрузка user/session/results/custom questionnaires в JSON backup.
- Очистка пользовательских данных из локального хранилища после выхода.
- Импорт backup файла на экране входа для восстановления профиля.
