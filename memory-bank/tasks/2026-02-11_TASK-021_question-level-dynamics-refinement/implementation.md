# Implementation Notes

## What changed
- `src/components/QuestionnaireStatsPanel.tsx`
  - Убрана табличная сводка как единственный формат.
  - Добавлены карточки по вопросам выбранного опросника:
    - вопрос + ID;
    - средний балл, последний балл, число попыток;
    - график истории изменения оценки (sparkline) по датам.
  - Динамика показывается только если у вопроса есть минимум 2 оценки.
  - Для вопросов с 1 оценкой выводится пояснение, что данных недостаточно для динамики.
- `src/utils/i18n.ts`
  - Добавлены ключи `questionStats.*` (RU/EN).
  - Уточнены `chart.timeline.*` тексты, что это общий дневной агрегат.
- `src/types/i18n.ts`
  - Добавлены типы ключей `questionStats.*`.
- `src/utils/formTranslationCoverage.ts`
  - Добавлены `questionStats.*` ключи в `student-dashboard`.

## Outcome
- Интерпретация `UR-034`/`UR-050` стала соответствовать пользовательскому ожиданию:
  - динамика наблюдается по конкретному вопросу;
  - только при наличии повторных ответов.
