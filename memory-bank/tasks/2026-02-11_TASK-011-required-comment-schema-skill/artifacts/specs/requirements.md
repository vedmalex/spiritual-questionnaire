# Requirements (Task-011)

## Source
Пользовательский запрос:
- добавить в схему вопроса обязательность письменного комментария,
- обновить skill `spiritual-questionnaire-architect`,
- выровнять требования skill под текущее приложение.

## Acceptance
1. Вопрос в JSON поддерживает `requires_comment`.
2. В редакторе можно управлять `requires_comment` для каждого вопроса.
3. При `requires_comment=true` переход к следующему шагу невозможен без комментария.
4. Skill обновлен и описывает совместимый JSON контракт.
