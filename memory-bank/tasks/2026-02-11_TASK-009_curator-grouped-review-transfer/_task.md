# TASK-009 — Curator Grouped Review Transfer

## Goal
Реализовать в панели куратора группировку заданий и полный цикл передачи: импорт ответов студентов -> проверка -> экспорт проверенных результатов для обратной передачи студентам.

## Linked Requirements
- `UR-053`: В панели куратора задания/ответы сгруппированы по опроснику и студенту.
- `UR-054`: Куратор может импортировать ответы студентов из файла для проверки.
- `UR-055`: Куратор может экспортировать проверенные результаты для передачи студенту.

## Deliverables
- Группировка результатов куратора по `questionnaire + student`.
- Импорт student answers из файла в панели куратора.
- Экспорт проверенных результатов (по группе и массово) для передачи студентам.
