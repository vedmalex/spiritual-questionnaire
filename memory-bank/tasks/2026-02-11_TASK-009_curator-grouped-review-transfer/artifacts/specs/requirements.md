# Requirements (Task-009)

## Source
Пользовательский запрос:
- задания у куратора должны быть сгруппированы,
- куратор может загружать ответы студентов,
- куратор выполняет проверку и выгружает результат для передачи студентам.

## Acceptance
1. В CuratorDashboard есть группировка по опроснику и студенту.
2. Куратор может импортировать ответы студентов из файла.
3. Куратор может экспортировать проверенные результаты:
   - массово,
   - по конкретной группе студента/опросника.
