# Requirements (Task-006)

## Source
Пользовательский запрос в чате:

> просмотр результатов должен выполняться по опроснику, и на дашборде общая оценка по опроснику словами и баллом

## Acceptance
1. Dashboard группирует результаты по опроснику.
2. В каждой группе отображается общая оценка в формате:
   - числовое значение `X/10`,
   - словесное описание по шкале USER-REQ.
3. Внутри каждой группы сохраняется доступ к истории прохождений.
