# TASK-006 — Dashboard Results by Questionnaire + Overall Score

## Goal
Обеспечить просмотр результатов в разрезе опросников и показывать на dashboard общую оценку по каждому опроснику словами и баллом.

## Linked Requirements
- `UR-013`: Dashboard с результатами и историей.
- `UR-043`: Просмотр результатов по опроснику + общая оценка словами и баллом.

## Deliverables
- Группировка результатов по `questionnaireId`.
- Карточка по каждому опроснику с:

  - датой последнего прохождения,
  - общей оценкой `X/10`,
  - словесным описанием оценки.
- История попыток внутри выбранного опросника.
