# Implementation

## Scope
- Реализован import assignment flow для неизвестных студентов при импорте в панели куратора.
- Добавлен выбор: назначить в существующую папку или создать новую папку для каждого нового студента.
- Добавлена ручная переупаковка (переназначение папки) уже импортированных групп в UI.

## What Was Changed
- Import flow:
  - `CuratorDashboard` анализирует import payload перед merge.
  - Для новых (ранее неизвестных) студентов при наличии папок открывается assignment modal.
  - В assignment modal для каждого студента:
    - `existing folder` selection,
    - либо `new folder` creation с вводом имени.
  - После подтверждения назначения сохраняются в sticky-маршрутизацию.
- Sticky routing and consistency:
  - Назначения сохраняются в `curatorResultFolders.studentFolderByKey`.
  - Новые работы известного студента автоматически отображаются в закрепленной папке.
- Manual repackaging:
  - В карточках групп добавлен селектор папки студента для ручного переноса между папками.
